# Biblical Number Significance

An interactive tool that helps users explore the significance of numbers in the Bible.

## Features

- Search for biblical meanings of numbers
- View scriptural references for each number
- Learn about numerical symbolism in scripture

## How to Use

1. Go to devinedigits.netlify.app
2. Enter a number in the search field
3. Click the "Search" button
4. View the meaning, description, and scripture references

## Technologies Used

- React
- JavaScript
- HTML/CSS
